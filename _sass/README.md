follow ITCSS (Inverted Triangle CSS) architecture. original structure:

```
______________
\____________/...SETTINGS           Far-reaching, Low specificity
 \__________/....TOOLS
  \________/.....GENERIC
   \______/......ELEMENTS(BASE)
    \____/.......OBJECTS
     \__/........COMPONENTS
      \/.........UTILITIES(TRUMPS)  localised, High specificity
```
